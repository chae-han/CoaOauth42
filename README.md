# CoaOauth42
42 클러스터 피씨에서 발생하는 iscsi target error를 학생 스스로 피씨 상태를 컨트롤 하여 문제를 해결할수 있도록 웹 으로 기능 제공하는 프로젝트. (맨날 타겟 에러 해주는거 매우 빡치기...때문...)

# 목표
* Oauth 개념을 확실하게 익히기

* 장고 활용 능력 기르기

* 파이썬을 파이썬 답게 쓰기 위한 고민을 하며 프로젝트 진행하기

* 이쁜 언어 파이썬을 이쁘게 잘 쓰기

* 기능 개성을 로그로 꾸준히 남길것
